Для создания шифрованного раздела надо ввести команду crzc с параметром размер раздела

sudo crzc 50G



$ sudo crzc 50G

Password: ---тут пароль текущего пользователя (для sudo)

Enter passphrase:  ---парольная фраза от раздела

Re-enter passphrase:   ---парольная фраза от раздела

$ 



Чтобы создать непривилегированного пользователя с домашним каталогом на шифрованном разделе, надо ввести команду crz_useradd, передав ей в качестве параметра имя пользователя.

sudo crz_useradd user2



$ sudo crz_useradd user2

Password: ---тут пароль текущего пользователя (для sudo)

New Password: ---Пароль создаваемого пользователя

Re-enter new Password:  ---Пароль создаваемого пользователя

passwd: password successfully changed for user2

$



Перезагрузиться, ввести пароль от шифрованного раздела залогиниться под непривилегированным пользователем.

Подключение шифрованного раздела после логина - crz
Отключение шифрованного раздела - crzu
crzg - тоже самое, что crz, но с запуском gnome-commander, в котором на левой панели содержимое шифрованного раздела

Удаление шифрованного раздела
sudo zfs destroy rpool/d
Последний параметр уточнить через zfs list
